### Context
OpenAI의 Clawdbot은 다양한 작업을 수행하도록 설계된 TypeScript 기반의 CLI 도구로, 프롬프트 최적화를 통해 작업 효율성을 높이는 것이 중요합니다. 특히 대규모 모델 사용 시 비용 절감과 정확한 정보 처리를 가능하게 하는 프롬프트 최적화는 필수적 과제입니다.

### Core
프롬프트 최적화를 위한 Clawdbot 설정 예시는 다음과 같습니다:
```typescript
// 프롬프트 구조 정의
const prompt = `작업을 세분화하고 각 단계에 필요한 세부사항을 제공하세요. 예를 들어, 파일 이름 변경, CSV 파싱, 레코드 필터링 등을 포함합니...``` 

이를 통해 작업 내용을 단계별로 구체화하여 처리하며, 각 작업을 별도의 프롬프트로 구성하여 모델이 무리 없이 작업할 수 있게 합니다.

### Insight
Clawdbot은 주어진 프롬프트를 통해 다양한 작업을 수행할 수 있습니다. 그러나 프롬프트의 양이 많아질수록 모델이 처리할 수 있는 최대 컨텍스트에 도달하게 됩니다. 이를 해결하기 위해 프롬프트를 압축하거나 변환하여 메모리에 저장하는 것이 효과적입니다. 이러한 방법은 OpenAI의 임베딩을 활용하여 해결할 수 있습니다.
